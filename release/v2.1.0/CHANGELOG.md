# 排班系统 v2.1.0 更新日志

## 新增功能

### 1. 移动端页面支持
- 提供了完整的移动端页面，访问地址：`/mobile/`
- 适配手机屏幕，优化触摸交互体验
- 移动端与电脑端数据完全同步

### 2. 安卓手机端支持
- 新增安卓客户端支持
- 支持原生安卓应用体验

### 3. 无课表导入流程优化
- 优化了无课表导入流程，操作更加简洁
- 在安卓手机中支持一键同步课表功能
- 提升了导入成功率与稳定性

### 4. 排班流程优化
- 支持对排班结果进行二次修改
- 支持排班表一键分享至 QQ 群
- 优化排班预览与确认流程

## 技术改进

- 前端路由优化，电脑端与移动端完全分离
- 后端支持多前端静态资源服务
- 构建脚本优化，支持同时构建两端前端

## 访问地址

| 端 | 地址 |
|---|---|
| 电脑端 | `http://localhost:8080/` |
| 移动端 | `http://localhost:8080/mobile/` |
