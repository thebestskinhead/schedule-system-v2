## 1. 排班管理

### 1.1 智能排班算法

**功能描述**: 系统根据无课表数据，自动计算最优排班方案

**核心能力**:
- ✅ 自动避开有课时间
- ✅ 均衡分配值班工作量
- ✅ 支持每时段多人值班
- ✅ 支持每人值班上限控制

**使用场景**:
- 每周定期排班
- 临时调整值班人员
- 节假日特殊排班

### 1.2 排班预览与调整

**功能描述**: 在确认排班前预览结果，支持手动微调

**操作功能**:
- 可视化排班表格
- 点击更换值班人员
- 冲突检测与提示
- 实时统计信息

### 1.3 排班参数设置

**可配置参数**:

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 当前周次 | 系统默认显示的周次 | 1 |
| 每时段人数 | 每个时段需要的值班人数 | 2 |
| 最少人数 | 低于此人数会警告 | 0 |
| 每人每天最多 | 防止一天值班太多次 | 1 |
| 每人每周最多 | 平衡周工作量 | 2 |
| 自动递增 | 确认后自动进入下一周 | 关闭 |

## 2. 无课表管理

### 2.1 手动录入

**功能描述**: 直接在系统中标记有课/无课时间

**界面特点**:
- 周一到周五，1-5节课表格
- 点击格子切换状态
- 绿色=无课(可排班)，红色=有课

### 2.2 教务系统导入

**功能描述**: 自动从教务系统抓取课程表并转换为无课表

**支持方式**:
- Cookie方式抓取
- 自动解析课程时间
- 智能转换为无课时间

**支持学校**: 可根据学校教务系统定制

### 2.3 Excel导入

**功能描述**: 通过Excel文件批量导入无课表

**模板格式**:
```
姓名 | 学号 | 周一-1 | 周一-2 | ... | 周五-5
张三 | 001  |   ✓    |   ✓    | ... |   ✗
```

## 3. 值班管理

### 3.1 我的值班

**功能描述**: 个人值班安排查看

**显示信息**:

- 值班状态
- 值班地点（如有）

### 3.2 值班确认

**功能描述**: 值班人员确认和完成状态跟踪

**状态流程**:
```

   ↓
已取消
```

### 3.3 值班统计

**功能描述**: 统计个人值班情况

**统计维度**:
- 本月值班次数
- 本学期值班次数
- 值班完成率
- 缺勤记录

## 5. 每周分工

### 5.1 分工发布

**功能描述**: 设置各部门本周值班的日期安排

**操作方式**:
1. 选择周次
2. 为每个部门勾选值班日期
3. 一键发布

**示例**:
```
第10周分工：
- 办公室：周一、周三、周五
- 竞赛部：周二、周四
- 项目部：周一、周四
- 科普部：周三、周五
```

### 5.2 分工查看

**功能描述**: 查看本周及历史分工安排

**查看维度**:
- 按周次查看
- 按部门筛选
- 历史记录追溯
