## 快速开始

### 1. 系统登录

1. 打开浏览器，访问系统地址
2. 输入学号和密码登录
3. 首次登录后建议修改密码

### 2. 完善个人信息

1. 点击右上角用户名，选择"个人资料"
2. 确认姓名、邮箱等信息正确
3. 如部门信息有误，联系管理员修改

### 3. 录入无课表

1. 进入"我的无课表"页面
2. 点击方格标记有课/无课时间
3. 绿色表示无课（可排班），红色表示有课
4. 点击"保存"按钮保存

## 无课表管理

### 手动录入

1. 进入"我的无课表"页面
2. 表格显示周一到周五，1-5节
3. 点击格子切换状态：
   - 🟩 绿色 = 无课（可排班）
   - 🟥 红色 = 有课（不可排班）
4. 点击"保存"提交

### 批量导入

1. 点击"批量导入"按钮
2. 选择导入方式：
   - **教务系统抓取** - 输入教务系统账号Cookie
   - **Excel导入** - 上传无课表Excel文件
3. 预览导入结果
4. 确认导入

### 导入爬虫使用

1. 进入"我的无课表" → "从教务系统导入"
2. 按照说明获取教务系统Cookie
3. 粘贴Cookie到输入框
4. 选择学年和学期
5. 点击"开始导入"
6. 系统将自动抓取课程表并转换为无课表

## 排班管理

> **注意**: 此功能需要排班管理权限

### 生成排班

1. 进入"排班管理"页面
2. 配置排班参数：
   - **周次** - 选择要排班的周次
   - **值班日期** - 选择星期几需要值班
   - **节次** - 选择每天的值班节数
   - **每时段人数** - 每个时段需要几人值班
   - **每人最多排班** - 限制个人值班次数
3. 选择排班部门
4. 点击"预览排班"

### 预览和调整

1. 系统根据无课表自动生成排班方案
2. 在预览表格中查看分配结果
3. 如需调整：
   - 点击单元格中的"更换"按钮
   - 选择其他可替换人员
4. 确认无误后点击"确认排班"

### 排班设置

在排班管理页面可以设置：

- **当前周次** - 首页默认显示的周次
- **自动递增** - 确认排班后自动进入下一周
- **每时段人数** - 默认值班人数
- **最少人数** - 低于此人数会警告
- **每人每天最多** - 防止一天排太多次
- **每人每周最多** - 平衡工作量
- **导出标题模板** - Excel导出标题格式

### 排班算法说明

系统排班算法遵循以下优先级：

1. **无课优先** - 只在无课时间排班
2. **均衡分配** - 尽量平均分配值班次数
3. **上限控制** - 不超过个人最大排班次数
4. **最少人数保证** - 确保每个时段有足够人员

## 值班管理

### 查看我的值班

1. 点击"我的值班"菜单
2. 查看本周及未来的值班安排
3. 包含日期、时段、状态

### 值班状态

- **待确认** - 刚排班，需要确认
- **已确认** - 已确认会按时值班
- **已完成** - 值班完成
- **已取消** - 值班取消

### 确认值班

1. 在"我的值班"页面找到值班记录
2. 点击"确认"按钮确认会按时值班
3. 值班完成后点击"完成"

## 用户管理

> **注意**: 此功能需要用户管理权限

### 查看用户列表

1. 进入"用户管理"页面
2. 查看所有用户信息
3. 可以按部门筛选

### 设置用户部门

1. 在用户列表找到目标用户
2. 点击"设置部门"
3. 选择部门：办公室/竞赛部/项目部/科普部
4. 确认保存

### 设置部门角色

1. 找到目标用户
2. 点击"设置角色"
3. 选择：
   - **部门管理员** - 可管理部门排班和成员
   - **部门成员** - 普通成员权限
4. 确认保存

## 临时权限

> **注意**: 此功能需要系统管理员或办公室管理员权限

### 授予临时权限

1. 进入"临时权限管理"页面
2. 在左侧表单填写：
   - **选择用户** - 被授予权限的用户
   - **权限** - 要授予的权限
   - **资源类型** - 全部/部门/特定
   - **过期时间** - 权限失效时间
   - **原因** - 授权原因（可选）
3. 点击"授予权限"

### 权限说明

| 权限 | 说明 | 适用场景 |
|------|------|----------|
| 排班管理(全部) | 管理所有部门排班 | 统筹协调 |
| 排班管理(部门) | 管理本部门排班 | 部门管理 |
| 用户管理(全部) | 管理所有用户 | 人事管理 |
| 用户管理(部门) | 管理本部门用户 | 部门管理 |
| 设置每周分工 | 设置各部门值班日期 | 统筹协调 |

### 资源类型说明

- **全部** - 对所有资源有效
- **部门** - 对用户当前部门有效（动态跟随）
- **特定** - 对指定资源有效

### 撤销权限

1. 在权限列表找到要撤销的权限
2. 点击"撤销"按钮
3. 确认撤销

### 清理过期权限

1. 点击"清理过期"按钮
2. 系统自动清理所有已过期的权限

## 常见问题

### Q1: 无法登录怎么办？

**可能原因**:
- 学号或密码错误
- 账号未注册
- 系统未初始化

**解决方法**:
1. 检查学号和密码
2. 如忘记密码，联系管理员重置
3. 首次使用需要完成系统安装向导

### Q2: 无课表导入失败？

**可能原因**:
- Cookie过期或错误
- 教务系统改版
- 网络问题

**解决方法**:
1. 重新获取Cookie
2. 检查网络连接
3. 尝试手动录入

### Q3: 排班时提示"无人可排"？

**可能原因**:
- 部门成员未录入无课表
- 约束条件太严格

**解决方法**:
1. 提醒部门成员录入无课表
2. 放宽约束条件（如增加每时段人数）
3. 减少排班天数

### Q4: 为什么看不到管理菜单？

**可能原因**:
- 没有管理权限
- 临时权限已过期

**解决方法**:
1. 查看"我的权限"确认权限状态
2. 联系管理员申请权限

### Q5: 如何修改部门？

部门信息只能由管理员修改：
1. 联系部门管理员或办公室管理员
2. 提供需要修改的部门信息
3. 管理员在"用户管理"中修改

### Q6: 值班时间冲突怎么办？

1. 联系排班管理员说明情况
2. 管理员可以在"排班管理"中调整
3. 或安排其他人员代班
